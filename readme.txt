Về vấn đề read file
có hai dạng read file: 
    1. readFileSync: là phương thức tuần tự gọi là phương thức 
        đồng bộ => sẽ đọc tuần tự 
    2. readFile: là phương thức bất đồng bộ => cần thêm hàm callback
    để xử lí dữ liệu khi đã được đọc xong
